About
=====
**Gaïa’s Guardian - The FISE Awakening** is an Open Source Game developped by Automatic Team from CESI Corp. 
Gaïa's Guardian is a platformer and action game set in a polluted cyberpunk world, where nature strives to reclaim its place against machines.
The player takes on the role of Gaïa, a protective entity capable of using nature-based powers to fend off mechanical forces and restore ecological balance.
